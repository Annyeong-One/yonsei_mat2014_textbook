
# abs() and round()

These functions perform numeric transformations.

---

## abs()

Returns absolute value.

```python
print(abs(-5))
````

Output

```
5
```

---

## round()

Rounds floating point numbers.

```python
print(round(3.14159,2))
```

Output

```
3.14
```

---

## Rounding Integers

```python
print(round(5.7))
```

Output

```
6
```
