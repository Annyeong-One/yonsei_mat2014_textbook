


# help() and dir()

These built-ins help explore Python objects.

---

## help()

Displays documentation.

```python
help(print)
````

---

## dir()

Lists available attributes.

```python
dir(str)
```

Example output

```
['capitalize','count','encode','find','lower']
```

---

## Practical Example

```python
x = "Python"

print(dir(x))
```

These tools help developers **discover available functionality**.
