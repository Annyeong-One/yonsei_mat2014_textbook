# Assignment Semantics

## Binding Semantics

### 1. Name Binding

```python
x = 42
# Creates/updates binding
# x → 42 object
```

## Multiple Targets

### 1. Unpacking

```python
a, b = 1, 2
# Creates two bindings
```

## Summary

- Assignment binds names
- Multiple assignment supported
- Unpacking available
