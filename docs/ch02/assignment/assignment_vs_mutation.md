# Assignment vs Mutation

## Assignment

### 1. Creates Binding

```python
x = [1, 2, 3]
x = [4, 5, 6]  # New binding
```

## Mutation

### 1. Modifies Object

```python
x = [1, 2, 3]
x.append(4)  # Same object
```

## Summary

- Assignment: new binding
- Mutation: modify in place
- Different operations
