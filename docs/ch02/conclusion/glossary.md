# Glossary

## A

**Assignment**: Binding name to object

**Aliasing**: Multiple names for one object

## B

**Binding**: Association between name and object

**Built-in**: Python's core namespace

## C

**Cell**: Storage for closure variables

**Closure**: Function with captured variables

**CPython**: Reference Python implementation

## F

**Free variable**: Non-local variable in closure

## G

**Garbage collection**: Automatic memory cleanup

**Global**: Module-level namespace

## I

**Identity**: Unique object ID

**Immutable**: Cannot change value

## L

**LEGB**: Lookup order (Local/Enclosing/Global/Builtin)

**Local**: Function-level namespace

## M

**Mutable**: Can change value

**Mutation**: In-place modification

## N

**Namespace**: Name-to-object mapping

**nonlocal**: Access enclosing variable

## O

**Object**: Python data entity

## R

**Reference**: Pointer to object

**Reference counting**: GC by tracking references

## S

**Scope**: Variable visibility region

**Shadowing**: Inner name hides outer
