# Parameter Mechanisms

## Positional

### 1. By Position

```python
def function(a, b, c):
    return a + b + c

function(1, 2, 3)
```

## Keyword

### 1. By Name

```python
function(a=1, b=2, c=3)
```

## Variable Length

### 1. *args and **kwargs

```python
def function(*args, **kwargs):
    pass
```

## Summary

- Positional
- Keyword
- Variable length
